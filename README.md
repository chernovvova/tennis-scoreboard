# Описание проекта

Веб-приложение, реализующее табло счёта теннисного матча. ТЗ к проекту: https://zhukovsd.github.io/java-backend-learning-course/projects/tennis-scoreboard/

# Используемые технологии

* Gradle
* Java Servlets, JSP
* HTML, CSS
* Hibernate, H2
* JUnit5
